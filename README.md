En este aplicativo se busca consumir un api publica para hacer el ejercicio de HTML, CSS y JS

HTML:
La estructura de HTML es básica con una diferencia y es que está dividida en cajas(div) para separar cada sección del
aplicativo y poder configurarlo más tarde en CSS

CSS:
Al haber separado el HTML en cajas(div) es más sencillo trabajarlo y darle la estética a cada parte de la pagina

JS:
Es un código sencillo que consume el api, la api Dragonball API guarda los personajes por numero 
...characters/1 y no por nombre ...characters/nombre, entonces en el código se definió una búsqueda
que trae el array con el parámetro “?name=” y encuentra el personaje.
